# 🌌 次元大战 · Dimensional War

一款纯前端（HTML/CSS/JS）的次元争霸放置冒险游戏，无需安装任何依赖，**直接用浏览器打开 `index.html` 即可游玩**。

## 玩法概览

### 1. 选择次元降临
五大次元世界任你选择，每个次元拥有专属的地图、怪物、技能体系与装备：

| 次元 | 风格 | 代表技能 |
|------|------|----------|
| ⚙️ 科技世界 | 机甲、轨道炮、纳米科技 | 等离子射击 / 轨道炮打击 / 机甲降临 |
| ⚔️ 修仙世界 | 御剑、符箓、渡劫飞升 | 御剑术 / 剑气纵横 / 飞升一击 |
| 🌃 赛博朋克世界 | 义体、黑客、霓虹都市 | 单分子刀 / 系统入侵 / 义体超频 |
| 🔮 西方魔法世界 | 魔法、巨龙、禁咒 | 火球术 / 陨石术 / 神罚禁咒 |
| 🎯 猎人世界 | 念能力、试炼、暗黑大陆 | 强化系重拳 / 圆之爆发 / 王之一击 |

### 2. 打怪升级
- 每天拥有 5 点行动点，探索地图区域消耗 1 点
- 击败怪物获得经验、金币，35% 概率掉落装备（普通→精良→稀有→史诗→传说 五档稀有度）
- 升级自动解锁次元专属技能（Lv.1/3/6/10/15），战斗为回合制，技能有冷却
- **同次元玩家受世界规则保护，无法互相战斗**，野外偶遇只会交换情报

### 3. 🌀 重叠区（大战前夕）
- 每周第 7 天开启次元大战，对阵双方在周初由世界意志公布
- **第 5~6 天，随机地图区域变成重叠区**，两个参战次元的空间在此交叠
- 进入重叠区有高概率遭遇敌方次元玩家，可以战斗——**胜利掠夺对方的金钱与装备**（异次元装备同样可以使用）

### 4. ⚔️ 次元大战（每周一次）
- 每周两个次元捉对厮杀，玩家若参战需代表本方迎战敌方三名勇士，胜场提升本方战力
- 大战胜负由双方次元总战力（全体居民 + 玩家战力×3 + 前哨战加成）决定
- **胜利**：获得世界意志奖励（金币、史诗/传说装备、大量经验），次元晋级下一轮
- **失败**：次元淹灭，所有降临者（包括你）**随机重生到其他存活次元**——等级、装备保留，技能体系转换为新次元
- 周而复始，直到只剩最后一个次元——**决出诸天万界最终胜利者**！

### 5. 📨 邀请系统 · 世界意志奖励
- 生成邀请码邀请好友降临你的世界
- 每邀请一名好友，**世界意志降下奖励**：金币、史诗/传说装备、全属性永久加成（可叠加）
- 好友会成为你方次元的强力居民，在次元大战中并肩作战、提升次元总战力

## 运行

```bash
# 方式一：直接双击打开
open dimensional-war/index.html

# 方式二：本地服务器
cd dimensional-war && python3 -m http.server 8000
# 浏览器访问 http://localhost:8000
```

进度自动保存在浏览器 localStorage 中，点击「重开」可开启新的轮回。

## 文件结构

```
dimensional-war/
├── index.html   # 页面结构（选择界面 / 主界面 / 战斗与事件弹窗）
├── style.css    # 暗色科幻主题样式
├── data.js      # 五大次元数据：地图、怪物、技能、装备、奖励表
└── game.js      # 核心逻辑：时间系统、战斗、重叠区PvP、次元大战、重生、邀请
```
