# 别慌 (Don't Panic)

一个原创的小游戏原型，零依赖（单个 `index.html`，浏览器直接打开就能玩，手机/电脑都支持）。

## 玩法 / 核心机制

你是一个光点，红色的"影子"会不断从四面八方追向你。

- **移动会让你的心跳（BPM）升高**；心跳越高，**所有追踪者的移动速度和刷新频率都会变快**——恐惧会喂养威胁本身。
- 按住 **屏息**（按钮 / 长按 空格 或 Shift）可以快速把心跳压回平静，但**屏息时无法移动**，等于把自己变成活靶子。
- 站着不动心跳也会缓慢回落，但比屏息慢很多。
- 被追上会扣一颗心（共 3 颗），且会造成一次心跳骤升（越慌越乱的正反馈），心跳恢复期间也更容易被下一个追上。
- 贴脸擦过但没被追上算一次"惊险闪避"，会计入分数。
- 目标：单纯地活得更久（生存秒数 = 分数），成绩存本地最佳记录。

核心张力在于「移动躲避」与「屏息平静」之间的取舍：一直躲会让世界越来越快、越来越乱；一直屏息又会让自己毫无反应能力。

## 怎么玩

直接用浏览器打开 `index.html` 即可，无需安装、无需服务器、无需构建：

```
dont-panic/index.html
```

（或者本地起个静态服务器打开，比如 `python3 -m http.server`。）

- 电脑：方向键 / WASD 移动，长按 空格 或 Shift 屏息。
- 手机/平板：触摸拖拽移动，长按右下角「屏息」按钮。

## 手机上怎么打开（不用任何 App）

这个分支还没合并进 main、也没部署到你现有的游戏服务器，最快的办法是用 GitHub Pages（免费、一次设置、几分钟内出一个手机能直接打开的网址）：

1. 手机或电脑浏览器登录 GitHub，进入本仓库 `zhaodadao123456-dev/CIYUANDAZHAN`
2. **Settings（设置）** → 左侧 **Pages**
3. Source 选 "Deploy from a branch"，Branch 选当前这个开发分支（或合并后选 `main`），文件夹选 `/ (root)`，点 **Save**
4. 等 1~2 分钟，手机浏览器打开：
   `https://zhaodadao123456-dev.github.io/CIYUANDAZHAN/dont-panic/`
