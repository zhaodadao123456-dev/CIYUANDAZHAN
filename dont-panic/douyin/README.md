# 《别慌》抖音小游戏版

和网页版（`dont-panic/index.html`）玩法完全一样，代码已经改成抖音小游戏要求的 `tt.*` API 写法（`game.js` + `game.json` + `project.config.json`），可以直接用抖音开发者工具打开。

**已经做完、可以直接用的部分：**
- 全部玩法/物理/心跳曲线逻辑照搬网页版，HUD、菜单、结算页全部改成手绘在 canvas 上（小游戏没有 DOM）。
- 触摸：拖拽移动躲避、右下角圆形按钮长按"屏息"，多指同时按（一只手指拖动+另一只手指屏息）都支持。
- 最佳记录用 `tt.setStorageSync` 存到本机。
- 已经在本地用 Node 模拟了一遍抖音小游戏运行环境（假的 `tt.*`/canvas），把菜单→开始→移动推高心跳→屏息压低心跳→撞击扣血→死亡结算→存档 全流程跑通、无报错。**但没有在真正的抖音开发者工具里点开过**——因为这台机器上没有装那个工具，也没有你的开发者账号，麻烦你第一次打开时留意一下有没有报错，截图告诉我就行。

## 有一步必须你自己来：注册开发者账号

这是唯一没法帮你连夜做完的部分，**因为需要你本人的身份信息去实名认证，别人没法代替**：

1. 电脑打开 <https://developer.open-douyin.com>（抖音开放平台），用你的抖音号登录
2. 按提示完成**开发者实名认证**（个人认证一般填身份证信息，几分钟到几十分钟审核）
3. 认证通过后，找"小游戏"或"小程序" → 创建应用，类目随便选一个休闲游戏类，创建完会给你一个 **AppID**（一串数字）
4. 打开这个仓库里的 `dont-panic/douyin/project.config.json`，把里面的 `"touristappid"` 换成你拿到的真实 AppID

## 怎么在手机上试玩

1. 电脑上装"开发者工具"：在 <https://developer.open-douyin.com> 里搜"开发者工具下载"，下载安装（Windows/Mac 都有）
2. 打开开发者工具，登录同一个账号，选择"导入项目"，项目目录选到本仓库的 `dont-panic/douyin/` 这个文件夹
3. 左边会出现一个手机模拟器画面——先在这里点点看能不能玩（能测出明显问题）
4. 工具界面上找"预览"按钮，会弹出一个二维码，**用手机上的抖音 App 扫这个码**，游戏就会在你手机里的抖音里直接打开、可以玩了（这一步不需要等官方审核，因为是你自己账号在预览自己的项目）
5. 如果以后想让所有人都能在抖音里搜到这个游戏（而不只是你自己账号能玩），要在开放平台里"提交审核"，字节官方审核通常要等几天，这个等待时间我也没法帮你缩短

## 如果嫌麻烦，想现在（今晚）就先玩到

抖音这条路必须先注册账号，没法跳过。如果只是想尽快玩到，网页版 `dont-panic/index.html` 不需要任何账号，手机浏览器直接能玩——具体怎么把它挂到一个手机能打开的网址上，看 `dont-panic/README.md` 里的说明（推荐用 GitHub Pages，几步设置，几分钟内就有一个手机能直接打开的链接）。
